# Как добавить видео

Чтобы добавить видео с Яндекс.Диска:

1. Скачайте видео по ссылке: https://disk.yandex.ru/i/PLFFBXpozl0Dqw
2. Переименуйте файл в `demo-video.mp4`
3. Поместите файл в папку `public/` вашего проекта
4. Видео автоматически появится на сайте

Альтернативный способ - использовать прямую ссылку:
- Можете загрузить видео на любой хостинг (YouTube, Vimeo, или CDN)
- Замените `/demo-video.mp4` в компоненте VideoPlayer на вашу ссылку

Поддерживаемые форматы: MP4, WebM, OGG
